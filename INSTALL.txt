Installation Instructions
*************************

Run the 'install' script in order to install sodalite.

 ENVIRONMENT VARIABLES for customization of the installation:
.__________________________________________________________________________.
|  ENV             |  DESCRIPTION               |  DEFAULT                 |
|------------------|----------------------------|--------------------------|
|- DATA_DIR        |  system data dir           |  /usr/bin                |
|- CONFIG_DIR      |  system config dir         |  /etc                    |
|- DATA_DIR        |  system data dir           |  /usr/bin                |
|- DATA_DIR        |  data dir                  |  /usr/share              |
|- DOC_DIR         |  documentation dir         |  /usr/share/doc          |
|- LOG_DIR         |  log dir                   |  /var/log                |
|- MIME_DIR        |  desktop entry dir         |  /usr/share/applications |
.__________________________________________________________________________.
